# ပုတီး စေတီ Counter

A mobile-friendly static web app for counting pagoda levels and completed pagodas.

## Features

- Count one level by tapping the golden pagoda.
- Track the current level and completed pagoda count.
- Automatically adds one completed pagoda when all levels are counted.
- Edit the counter name, note/prayer text, levels per pagoda, target pagodas, and tap step.
- Save progress locally in the browser.
- Export a PNG progress report.

## Run

Open `index.html` in a browser.
