# ပုတီး စေတီ Counter

A mobile-friendly static web app for counting ပုတီး progress using the formula from the reference image:

```text
16 levels x 10 rounds x 108 beads = 160 rounds / 17,280 beads
```

## Features

- Count beads by tapping the main counter.
- Track the current level, round inside the level, and current bead.
- View all 16 pagoda levels with 10 rounds per level.
- Edit the counter name, mantra text, level count, rounds per level, beads per round, and tap step.
- Save progress locally in the browser.
- Export a PNG progress report.

## Run

Open `index.html` in a browser.
